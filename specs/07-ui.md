
# 07 - UI Frontend (Chat Interface)

## Overview
A modern, responsive, and user-friendly React-based chat interface for interacting with the Tata Nexon Agentic RAG Chatbot.

## Purpose
Provide an intuitive chat experience where users can ask questions about the Tata Nexon car and receive high-quality, streamed, and well-cited answers from the backend agent.

## Tech Stack
- **React 18+** with **Vite** (build tool)
- **TypeScript**
- **Tailwind CSS** for styling
- **EventSource (SSE)** for streaming responses
- **Axios** or native fetch for REST calls
- **React Markdown** for rendering answers and citations

## Core Features

### Chat Experience
- Modern chat UI similar to ChatGPT / Claude
- User messages on the right, assistant messages on the left
- Real-time streaming of answers (token by token)
- Proper rendering of citations/sources with clickable links
- Display confidence score and grounding status
- Loading / thinking indicators

### Session Management
- "New Chat" button to start a fresh conversation
- Sidebar showing list of previous conversations
- Support for multiple chat sessions using `thread_id`
- Ability to rename or delete conversations

### Admin Section (Protected)
- Document upload and ingestion interface
- View list of ingested documents
- Force re-ingest option
- Basic admin dashboard

## Non-Functional Requirements
- Fully responsive (mobile + desktop)
- Dark mode by default (with light mode toggle)
- Smooth animations and transitions
- Excellent accessibility (ARIA labels, keyboard navigation)
- Fast loading and smooth streaming experience
- Proper error handling with user-friendly messages

## API Integration
- **POST** `/chat` with optional `stream=true`
- Use **Server-Sent Events (SSE)** when streaming is enabled
- Send `thread_id` for conversation continuity
- Graceful handling of guardrail blocks and errors
